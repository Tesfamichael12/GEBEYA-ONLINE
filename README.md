## 📖 About the Project

GEBEYA ONLINE is a landing page for an online market store. It is designed to provide users with a seamless shopping experience. The project is built using:

- HTML
- CSS
- JavaScript

## 🚀 Features

- User-friendly interface
- Responsive design
- Easy navigation
